# Genres Endpoints

| Method | Endpoint      | Description         |
|--------|---------------|---------------------|
| GET    | Read Genres   | Read all genres     |

### Example

#### Get All Genres
This endpoint is used to get all Genres found on the system.

**GET /genres**
```json
[
  {"id": 1, "name": "Heavy Metal"},
  {"id": 2, "name": "Thrash Metal"}
]
```
